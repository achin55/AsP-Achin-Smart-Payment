## Smart Contracts

This folder contains Solidity smart contracts used in the AsP project.

- AsPToken.sol – ERC-20 utility token
- Presale.sol – Presale contract logic
- AsPTokenBNB.sol - mainnet logic for BNB chain
- PaymentRouter.sol - payment logic i.e fee reciever

Contracts are written using Solidity and OpenZeppelin standards.
